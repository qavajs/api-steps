# @qavajs/steps-api
qavajs steps to work with web API

## Installation

`npm install @qavajs/steps-api`

## Configuration
```javascript
module.exports = {
    default: {
        require: [
            'node_modules/@qavajs/steps-api/index.js'
        ],
        api: {
          logPayload: false //display payload logs - default true
        }
    }
}
```
